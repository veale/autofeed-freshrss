Bundled Playwright (default)
  • ✅ Runs in-process. No external service needed.
  • ✅ Full XHR/fetch capture — finds JSON APIs and GraphQL ops.
  • ⚠ No anti-bot evasion. Cloudflare Turnstile will block this.
  • ⚠ ~1 GB RAM per concurrent browser.
Scrapling Stealth
  • ✅ In-process. Bypasses Cloudflare Turnstile automatically.
  • ✅ Fingerprint spoofing, WebRTC leak blocking, real-Chrome mimicry.
  • ⚠ Parse-only — does not capture XHR responses. If a site's best feed is a JSON API, discovery through Stealth won't find it.
  • ⚠ Slower than Bundled (solve_cloudflare adds ~10–30 s per fetch).
External Playwright server
  • ✅ Offload browser load to a dedicated container (scales better).
  • ✅ Full XHR capture via remote CDP.
  • ⚠ Needs playwright run-server running somewhere reachable; set Playwright server URL below.
  • ⚠ No stealth — same anti-bot limitations as Bundled.
Browserless
  • ✅ Hosted Chromium-as-a-service (self-hosted or cloud).
  • ✅ Full XHR capture via CDP.
  • ⚠ Paid tier for cloud; container for self-hosted.
  • ⚠ No automatic stealth; available via plugins on paid tiers.
Scrapling-serve
  • ✅ Scrapling's stealth capabilities via HTTP API.
  • ⚠ Separate container to run and maintain.
  • ⚠ Parse-only — no XHR capture (same as in-process Stealth).
